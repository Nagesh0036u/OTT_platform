# 🎬 Mini Netflix

A content-based movie recommendation system built using Machine Learning and Streamlit.

## DEMO LINK:
https://mini-netflix-rlfmvmrqvlvwqddff8us43.streamlit.app/

## Features

- Recommend 5 similar movies instantly
- Live movie posters from TMDB API
- Content-based recommendation using cosine similarity
- Fast cached dataset loading
- Responsive Netflix-inspired UI
- Graceful handling of missing posters and API errors

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Requests
- TMDB API
